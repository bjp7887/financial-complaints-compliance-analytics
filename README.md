# Financial Complaints Compliance Analytics Dashboard

## Project Overview

This project analyses consumer financial complaint data to identify complaint trends, high-risk financial products, company response performance, and potential compliance concerns.

The objective is to demonstrate how data analytics can support regulatory monitoring, operational improvement, and customer complaint management in the financial services sector.

## Dataset

The dataset was sourced from the Consumer Financial Protection Bureau Consumer Complaint Database.

Source: https://www.consumerfinance.gov/data-research/consumer-complaints/

## Business Questions

1. Which financial products receive the highest number of complaints?
2. Which companies have the highest complaint volumes?
3. Are companies responding to complaints in a timely manner?
4. What are the most common complaint issues?
5. Which areas may indicate operational or compliance risk?

## Dashboard Features

- Complaint volume analysis
- Monthly trend analysis
- Product-level complaint breakdown
- Company-level complaint ranking
- Timely response monitoring
- Compliance risk indicators
